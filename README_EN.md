# ETL Sales Project

## Description
A simple end-to-end ETL project on a sales dataset (`Sample Superstore`).  
- Data import and cleaning were done in Python using Pandas  
- Cleaned data is saved in `data/processed`  
- Sales and profit analysis were performed in a Jupyter Notebook  
- SQL procedures (business logic) are in the `sql/` folder

## Project Structure

data/
raw/ # Original dataset
processed/ # Cleaned dataset

notebooks/ # ETL notebook and analysis
sql/ # SQL procedures for business logic
README.md
.gitignore


## How to run the project

1. Open the notebook in `notebooks/`  
2. Make sure the CSV files are in the `data/processed/` folder  
3. Run the cells to see the data cleaning and analysis

## Technologies used
- Python (Pandas)  
- Jupyter Notebook  
- SQL (SQLScript)
